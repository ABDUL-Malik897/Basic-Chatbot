# Basic-Chatbot
A simple rule-based chatbot created using Python. 

The chatbot can respond to greetings, basic questions, jokes, and also tell the current date and time.

## Features
- Greeting conversations
- Answers basic questions
- Tells current time and date
- Simple command-based interaction
- Beginner-friendly project

## Technologies Used
- Python
- Datetime Module

## Concepts Used
- if-elif Statements
- Loops
- Functions
- Input/Output
- Datetime Handling

## Commands Supported

- hello
- hi
- how are you
- what is your name
- what is python
- tell me a joke
- time
- date
- day
- help
- bye

## How to Run

1. Save the file as `basic_chatbot.py`
2. Open terminal or command prompt
3. Run the following command:

```bash
python basic_chatbot.py
```

## Example Conversation
```
You: hello
Bot: Hello! How can I help you today?

You: time
Bot: Current Time is 08:45 PM
```
## Project Objective

The objective of this project is to learn the basics of chatbot development using Python and understand conditional statements and loops.
